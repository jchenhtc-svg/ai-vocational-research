## 七、SmartPLS 4 分析計畫

### 7.1 樣本規劃

| 項目 | 數值 |
|:----:|:----:|
| 最低樣本（Cohen, f²=0.15, 80% power） | n ≥ 118 |
| 安全目標 | n ≥ 200 |
| 目標樣本 | n ≥ 300（含可能無效問卷） |

### 7.2 分析步驟

| 步驟 | SmartPLS 4 功能 | 檢查指標 |
|:----:|-----------------|:---------:|
| **Step 1: Measurement Model** | PLS Algorithm | Loadings > 0.708, AVE > 0.5, CR > 0.7, HTMT < 0.85 |
| **Step 2: Structural Model (Direct)** | Path Coefficients | β, t, p, CI (BCa, 5,000 subsamples) |
| **Step 3: Moderation** | Two-stage approach | β 交互項, p, f² |
| **Step 4: Mediation** | Specific Indirect Effects | 間接效果 CI ≠ 0, VAF |
| **Step 5: Model Fit** | SRMR, NFI | SRMR < 0.08 |
| **Step 6: Predictive Power** | Blindfolding (d=7) | Q² > 0 |

### 7.3 IPA 分析（在 SmartPLS 中執行）

| IPA 維度 | 操作化方式 | 來源 |
|:---------:|-----------|:----:|
| **重要性（Importance）** | AIPACK 各構面到整體 AIPACK 的**總效果**（direct + indirect）| SmartPLS 4 IPMA |
| **表現度（Performance）** | 各構面的**潛在變項平均分數**（轉換為 0-100 分） | SmartPLS 4 IPMA |
| **四象限** | 高重要高表現（維持）、高重要低表現（優先改善）、低重要高表現（過度投入）、低重要低表現（低優先）| — |
| **目標構面** | AIPACK 總分 | — |

## 九、研究限制

| 限制 | 說明 | 緩解方式 |
|:----:|------|----------|
| 橫斷設計 | 因果推論有限 | SmartPLS 僅能驗關聯性，無法確定因果方向 |
| 自陳量表 | 共同方法變異 | Harman 單因子檢定、CMV 控制法 |
| 單一群體 | 僅電機電子群教師 | 建議後續擴展至其他職科 |
| 樣本數 | 即使 n=200 仍有限 | 小樣本須使用 PLSpredict 驗證預測穩定性 |
| Mooring 雙重角色 | 統計上沒有單一檢定可以同時驗證 mediator 與 moderator | 分別跑兩個模型比較 |
