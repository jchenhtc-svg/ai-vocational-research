# AI 融入教學研究：技術型高中電機電子群教師 AI 輔助教學轉型

## 研究概述

本研究以 Push-Pull-Mooring（PPM）模型探討技術型高中電機電子群科教師 AI 輔助教學轉型之影響機制與專業能力建構歷程，採三階段序列混合研究設計。

### 研究一：量化 PPM 驗證
- 探討 Push（教學困境）、Pull（AI 吸引力、情緒能力、同儕影響、TTF）與 Mooring（促進型/阻礙型）對轉換意願之影響
- 檢驗 Mooring 之干擾與中介雙重角色
- 分析工具：SmartPLS 4（PLS-SEM），目標樣本 n ≥ 200

### 研究二：量化 AIPACK 建構與 IPA 診斷
- 探討轉換意願對 AIPACK 七大構面（AI-TK, AI-TCK, AI-TPK, AI-TPACK, XK, i-CK, i-PD）之影響
- 以 IPA（重要性-表現度分析）診斷優先介入構面
- 分析工具：SmartPLS 4（PLS-SEM + IPMA）

### 研究三：質性深度訪談
- 針對 IPA 四象限典型個案進行半結構式訪談（8-10 位）
- 探討象限歸因之情境脈絡與轉化路徑
- 分析工具：主題分析法（Braun & Clarke, 2006）

## 分析工具

- **SmartPLS 4**：PLS-SEM 路徑分析、Mooring 雙重角色檢驗、IPMA
- 樣本目標：n ≥ 200（PLS-SEM 小樣本優勢，最低可至 n = 118）
- 關鍵設定：Bootstrapping 5,000 subsamples、BCa CI、Two-stage moderation、HTMT < 0.85
